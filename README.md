## Installation

Clone the repo and run 

```bash
ln -s <repo> /usr/local/bin/
```

Where <repo> is the location of journalz_ro folder you cloned.

e.g. I cloned the repo into my home directory

```bash
ln -s ~/journalz_ro /usr/local/bin/
```
